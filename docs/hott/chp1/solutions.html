1.2 Proof Using Type Theory and Lambda Calculus
Step 1: Deﬁnitions
Deﬁne the functions f , g, and h as lambda expressions:
• f ≡ λx.f(x) where f : A → B
• g ≡ λy.g(y) where g : B → C
• h ≡ λz.h(z) where h : C → D

• g ◦ f:
The composition g ◦ f : A → C is deﬁned as:
g ◦ f ≡ λx.g(f(x))
This means that for an input x ∈ A, we ﬁrst apply f to get an
element of B, and then apply g to get an element of C.

To compose h with g ◦ f , we have:
h ◦ (g ◦ f) ≡ λx.h(((g ◦ f )(x)))

h ◦ (g ◦ f) ≡ λx.h((g(f(x))))

• h ◦ g:
The composition h ◦ g : B → D is deﬁned as:
h ◦ g ≡ λy.h(g(y))
– Here, for an input y ∈ B, we apply g to get an element of C,
then apply h to get an element of D.

(h ◦ g) ◦ f ≡ λx.(h ◦ g)(f(x))

(h ◦ g) ◦ f ≡ λx.h(g(f(x)))
